# 🏫 대학 홈페이지 웹 서비스 구축

## 📌 프로젝트 정보
- 기간 : 2024.04 ~ 2024.06  
- 기관 : 동양미래대학교  
- 참여 형태 : 2인 팀 프로젝트 (팀장)  
- 기여도 : 80%

## 📖 프로젝트 소개
신입생과 재학생을 위한 대학 홈페이지 웹 서비스를 Java 및 JSP 기반으로 구축한 프로젝트입니다.  

학과 소개, 학사 일정, 공지사항, 캠퍼스 맵, 로그인 시스템 등을 구현하여  
사용자가 학교 정보를 직관적으로 확인할 수 있도록 구성하였습니다.

## 🧑‍💻 담당 역할
- 팀장 역할 수행 및 프로젝트 일정 관리  
- 전체 페이지 구조 및 메뉴 구성 설계  
- 지도 API 연동 및 캠퍼스 위치 시각화 구현  
- 로그인 기능 및 DB 연동 처리  
- 프론트엔드와 백엔드 연동 구조 설계  

## 📚 Lesson Learned
- JSP 기반 웹 서비스 전반적인 구조 이해  
- API 연동을 통한 지도 시각화 기능 구현 경험  
- 세션 관리 및 사용자 인증 처리 실무 경험  
- 소규모 팀 프로젝트 리딩 및 협업 역량 강화
